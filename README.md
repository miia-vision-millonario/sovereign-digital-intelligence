## 🚀 Instalación

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/miia-vision-millonario/sovereign-digital-intelligence.git
   cd sovereign-digital-intelligence
   ```

2. **Ejecutar localmente:**
   - Opción A: Usar Python
     ```bash
     python -m http.server 8000
     ```
   - Opción B: Usar Node.js (con http-server)
     ```bash
     npx http-server
     ```
   - Opción C: Usar VS Code Live Server (extensión)

3. **Acceder a la aplicación:**
   ```
   http://localhost:8000
   ```

## 📋 Cómo Agregar Nuevos Activos

Edita `assets_registry.json` y agrega una nueva entrada:

```json
{
  "asset_XXX": {
    "id": "asset_XXX",
    "title": "Tu Producto",
    "niche": "Categoría",
    "price": 97,
    "description": "Descripción breve",
    "features": [
      "Feature 1",
      "Feature 2"
    ]
  }
}
```

El catálogo se actualizará automáticamente en la página.
